Variáveis

Em programação, variáveis são usadas para armazenar valores que podem mudar ao longo do tempo. Elas são essenciais para guardar informações, fazer cálculos, controlar sensores ou mesmo imprimir mensagens no console.

Para criar uma variável, vá até a categoria Variáveis no menu lateral e clique em Criar variável.... Escolha um nome claro e significativo, pois isso evita erros e torna o programa mais fácil de entender.

Após criada, sua variável aparecerá em Variáveis criadas e poderá ser usada nos blocos para definir, alterar ou acessar seu valor.

Existem diferentes tipos de variáveis disponíveis:

  • Booleanas: representam verdadeiro ou falso. Úteis para verificações ou condições.
  • Numéricas: armazenam números inteiros, decimais ou aleatórios, comuns em leituras de sensores e cálculos.
  • Texto: armazenam frases ou palavras, ideais para mostrar mensagens ou nomes.
  • Listas: armazenam vários valores em uma mesma variável. Muito útil para organizar dados.

Na robótica, variáveis são muito utilizadas para armazenar leituras de sensores como temperatura, distância, luz, entre outros. Isso permite tomar decisões ou ajustar comportamentos com base nos dados do ambiente.

Laços de repetição (Loops)

Loops são estruturas que permitem executar um mesmo conjunto de comandos várias vezes. Eles são úteis para automatizar tarefas repetitivas, como contar, exibir mensagens, ou manipular listas.

Na plataforma, os blocos de repetição estão disponíveis na categoria Controle e incluem diferentes tipos de laços com finalidades variadas.

1. Repetição com número fixo

O bloco repita X vezes executa o conjunto de comandos um número específico de vezes. Ideal para contagens simples ou ações repetidas.

2. Repita enquanto verdadeiro

O bloco repita enquanto verdadeiro executa os comandos de forma contínua, sem parar. É muito usado em projetos de robótica, onde o programa precisa rodar constantemente enquanto o sistema estiver ligado.

Neste exemplo, a variável contador é incrementada a cada segundo, e seu valor é impresso no console repetidamente.

3. Repetição enquanto uma condição for verdadeira

O bloco repita enquanto continua executando os comandos enquanto a condição definida for verdadeira.

Neste exemplo, a variável numero é incrementada até chegar a 5. O loop para quando a condição numero < 5 não for mais verdadeira.

4. Repetição com contador personalizado

O bloco contar com i de X até Y por Z permite maior controle sobre a repetição, incluindo o valor inicial, final e o passo.

Neste exemplo, imprimimos a tabuada do 2, multiplicando o valor de i a cada repetição.

5. Repetição para cada item de uma lista

O bloco para cada item na lista percorre todos os elementos de uma lista. Ideal para processar ou exibir vários dados.

Aqui temos uma lista de nomes. O loop imprime um nome por vez até que todos sejam exibidos.

6. Encerrar laço com condição

O bloco encerra o laço permite interromper a execução do loop antes de ele completar todas as repetições.

Neste exemplo, contamos de 1 até 10, mas encerramos o laço quando o contador atinge o valor 6.
